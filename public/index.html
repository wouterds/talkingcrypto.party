Currencies API

We have some REST API endpoints to get crypto currency information, updated minutely. Example usage:

  • https://api.talkingcrypto.party/currencies
  • https://api.talkingcrypto.party/currencies?include=price
  • https://api.talkingcrypto.party/currencies/1
  • https://api.talkingcrypto.party/currencies?symbol=BTC
  • https://api.talkingcrypto.party/currencies?symbol=BTC,LTC,ETH&include=price
